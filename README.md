# event-manager-service
Web Api для сервиса управления мероприятиями

Для запуска проекта 
1. Перейдите в корневую папку решения
2. Запустите dotnet build
3. Перейдите в папку проекта  cd .\EventManagerService\
4. Используйте dotnet run -lp http (по умолчанию используется порт 5214) или dotnet run
5. После успешного запуска откройте http://localhost:port/swagger/index.html для доступа к swagger, вместо port подставьте порт из вывода dotnet run
